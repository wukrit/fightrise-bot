# Zero to Beta: Complete Setup Guide

This guide walks you through setting up FightRise from a fresh clone to a running tournament bot.

## Prerequisites

- Node.js 18+ installed
- Docker and Docker Compose installed
- A Discord account with a server you admin
- A Start.gg account (for tournament management)

## Estimated Time

**30-45 minutes** for initial setup

## Steps Overview

1. Clone and install dependencies
2. Configure environment variables
3. Set up Discord bot
4. Set up Start.gg API
5. Start infrastructure (PostgreSQL, Redis)
6. Deploy Discord commands
7. Run the bot

---

## Step 1: Clone and Install

```bash
git clone https://github.com/wukrit/fightrise-bot.git
cd fightrise-bot
npm install
```

## Step 2: Configure Environment Variables

FightRise uses DMNO for environment management. Copy the example files:

```bash
cp .env.example .env
cp .dmno.env.local.example .dmno.env.local
```

Edit `.dmno.env.local` with your secrets:

```bash
# Generate a NextAuth secret
openssl rand -base64 32
# Add the output to NEXTAUTH_SECRET

# Add your Discord bot token
DISCORD_TOKEN=your_bot_token

# Add your Start.gg API key
STARTGG_API_KEY=your_startgg_api_key
```

See the [DMNO Guide](/fightrise-bot/guides/dmno-setup/) for detailed environment configuration.

## Step 3: Set Up Discord Bot

1. Go to [Discord Developer Portal](https://discord.com/developers/applications)
2. Create a new application
3. Go to the **Bot** tab:
   - Click "Reset Token" and copy it
   - Enable "Message Content Intent" under Privileged Gateway Intents
4. Go to **OAuth2** > **General**:
   - Add redirect: `http://localhost:4000/api/auth/callback/discord`
5. Go to **OAuth2** > **URL Generator**:
   - Select scopes: `bot`, `identify`, `guilds`
   - Select permissions: `Manage Threads`, `Send Messages`, `Embed Links`, `Use Slash Commands`
   - Copy the generated URL and invite the bot to your server

See [Discord Setup Guide](/fightrise-bot/guides/discord-setup/) for detailed instructions.

## Step 4: Set Up Start.gg API

1. Go to [Start.gg Developer](https://start.gg/admin/profile/developer)
2. Create a new API token
3. Copy the token (it won't be shown again!)

See [Start.gg Setup Guide](/fightrise-bot/guides/startgg-setup/) for detailed instructions.

## Step 5: Start Infrastructure

FightRise requires PostgreSQL and Redis. Use Docker:

```bash
npm run docker:infra
```

This starts:
- PostgreSQL on localhost:5432
- Redis on localhost:6379

Push the database schema:

```bash
npm run docker:db:push
```

## Step 6: Deploy Discord Commands

Before running the bot, deploy the slash commands:

```bash
npm run docker:deploy
```

## Step 7: Run the Bot

```bash
# Development mode (recommended)
npm run dev

# Or run just the bot
npm run run:bot
```

## Verification

1. Check the bot is online in Discord
2. Try `/link-startgg` to link your Start.gg account
3. Try `/tournament status` to see tournament info

## Troubleshooting

### Bot not responding to commands

- Ensure commands were deployed: `npm run docker:deploy`
- Check bot has correct permissions in Discord server settings

### Database connection errors

- Verify Docker is running: `docker ps`
- Check DATABASE_URL in .env matches your Docker setup

### Redis connection errors

- Verify Redis is running: `docker ps`
- Check REDIS_URL in .env
