# Player Quickstart

This guide helps players get started with FightRise.

## Prerequisites

- Discord account
- Start.gg account (for tournament registration)
- FightRise bot in your tournament's Discord server

## Getting Started

### Step 1: Link Your Start.gg Account

```
/link-startgg
```

1. Run the command
2. Click the link in the response
3. Authorize with Start.gg
4. Return to Discord - your account is linked!

### Step 2: Register for Tournament

```
/register
```

1. Run the command
2. Enter the tournament slug (from TO or event page)
3. Wait for TO approval (if required)

### Step 3: Find Your Matches

```
/my-matches
```

Shows your upcoming matches in the tournament.

## During Matches

### When Your Match is Called

1. You'll receive a Discord ping in a match thread
2. The thread shows your opponent and round info

### Checking In

**Important:** Check in by clicking the **button in the match thread**, not using the `/checkin` command.

1. Click the check-in button in the match thread
2. Do this within the check-in window (usually 10-15 minutes)
3. Wait for your opponent to also check in

### Reporting Scores

**Important:** Report scores using the **buttons in the match thread**, not using the `/report` command.

When both players are checked in:

1. Click "Player X Won" for a quick 2-0/3-0 win
2. Or use the dropdown for detailed scores (2-1, 3-2)
3. Your opponent must confirm the result
4. If they don't respond within the window, the result is auto-confirmed

### Disputing Results

If the reported score is wrong:

1. Click "Dispute" on the result
2. Contact your TO in the thread or via DM
3. Provide details about the correct score

## Common Issues

### Can't find /link-startgg command

- The bot may not be responding
- Contact your TO to verify the bot is working

### Check-in button not working

- Make sure you're clicking the button in the thread, not using a command
- Check you're within the check-in time window

### Opponent not confirming

- Wait for the auto-confirmation timer
- If urgent, ping your TO in the thread

### Match thread not found

- Use `/my-matches` to see your active matches
- Check Discord notifications aren't blocked
